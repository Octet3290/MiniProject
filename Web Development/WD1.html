Introduction To Web Development

Web development refers to the building, creating, and maintaining of websites.

Types of Web development

1. FRONTEND DEVELOPMENT - This is the user interface of website.
2. BACKEND DEVELOPMENT - This is the server side of the website. It is used to store and arrange data.
